# Changelog

## 1.0.0

Initial release.

- **Witnesses react.** Living NPCs (citizens and police) who actually see a dead NPC report it: citizens get the
  police called, police enter the game's real "Investigating" pursuit. Bodies hidden from sight (dumpster, indoors,
  underwater, behind cover) are not seen, so the gameplay is "hide your bodies".
- **Heavier corpses.** A carried corpse is made heavier (configurable, default ~5x) so disposal has weight and stress.
- **Configurable realism.** Hunt the killer / Scene investigation / Escalating, line-of-sight on/off, which NPC types
  react, pursuit level, reaction delay, cooldowns, weight, and more - in the in-game Mod Manager / Phone App or
  `MelonPreferences.cfg`.
- Host-authoritative; multiplayer is off by default (`EnableInMultiplayer`) until tested with two players.
